## 🛠️ Tech Stack

This project is built using modern, cutting-edge web technologies:
- **Framework:** [Next.js 14+ (App Router)](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **3D Graphics:** [Three.js](https://threejs.org/) & [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/getting-started/introduction)
- **Database / Backend:** [Supabase](https://supabase.com/) (For contact form handling)
- **Deployment:** Vercel

## ✨ Key Features
- **Immersive 3D Experience:** Custom interactive WebGL Starfield background that reacts to mouse movement.
- **Enterprise SEO:** Dynamically generated OpenGraph meta tags, fully optimized JSON-LD Structured Data, and lazy-loading for perfect Lighthouse performance scores.
- **Flawless Accessibility:** Full ARIA compliance across all interactive elements, dynamic form validation, and screen-reader support.
- **Serverless Form Handling:** Secure, scalable contact form powered by Supabase.

## 💻 Running Locally

To get a local copy up and running, follow these simple steps:

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/avi735/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Set up environment variables (copy `.env.example` to `.env.local` and add your Supabase credentials).
5. Start the development server:
   ```bash
   npm run dev
   ```
6. Open [http://localhost:3000](http://localhost:3000) in your browser.
